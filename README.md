## Usage
 1. eAutoLauncher.exe を実行する
 2. 実行したい項目を1-5から選ぶ

## Support OS
 - Windows  
 (以下のOSは非積極的なサポートです。)
 - macOS
 - Linux
 
## Need for install Python Library
 - sqlite3
 - requests
 - selenium
 - BeautifulSoup
 - lxml
